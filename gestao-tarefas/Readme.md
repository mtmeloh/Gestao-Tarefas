📋 Gestão de Tarefas API

API REST para gerenciamento de tarefas desenvolvida com Spring Boot, incluindo testes unitários e documentação via Swagger.

🚀 Tecnologias utilizadas
Java 17+
Spring Boot
Spring Web
Spring Data JPA
PostgreSQL 
JUnit (testes unitários)
Swagger / OpenAPI
Maven


Funcionalidades:
Criar tarefas
Listar tarefas
Buscar tarefa por ID
Atualizar tarefas
Remover tarefas
